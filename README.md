# Прайс-лист

PWA-приложение для просмотра прайс-листа на монтажные работы.

## Демо

👉 https://getto-dev.github.io/price/

## Категории услуг

- 🔥 Отопление
- 🌡️ Тёплый пол
- 💧 Водоснабжение
- 🏠 Котельные
- 💨 Дымоходы
- 🚿 Канализация
- 🔧 Прокладка труб
- 💎 Водоочистка
- ⚙️ Автоматика
- 🔨 Штробление
- 🛠️ Сервис
- 🚽 Сантехника

## Возможности

- 📱 Устанавливается как приложение (PWA)
- 🌙 Тёмная тема
- 🔍 Поиск по услугам
- 📋 Детальное описание работ
- 💰 Актуальные цены

## Разработка

```bash
bun install
bun run dev
```

## Сборка

```bash
bun run build
```

## Данные

Прайс-лист хранится в [price-data](https://github.com/getto-dev/price-data) репозитории.
